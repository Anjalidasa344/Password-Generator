# Password Generator in Python 🔐

## Overview

Password Generator is a simple Python-based application that generates random passwords of user-defined lengths. The program creates strong passwords by combining:

- Lowercase letters
- Uppercase letters
- Numbers

This project is useful for learning Python fundamentals, randomization techniques, and string manipulation.

---

## Features

✅ Generate multiple passwords at once

✅ User-defined password lengths

✅ Automatic inclusion of numbers

✅ Automatic inclusion of uppercase letters

✅ Simple command-line interface

✅ Lightweight and beginner-friendly

---

## Technologies Used

- Python 3
- Random Module
